h1. Gédéon Intranet

Gédéon est un système de gestion de documents techniques client-serveur. Gédéon Intranet est sa version web.

La société Artesys International est spécialisée dans la gestion électronique de documents techniques, notamment orientée vers l’industrie. Depuis plusieurs années, son produit phare était Gédéon, un système d’origine client-serveur dont la vocation était de faciliter l’accès à la documentation d’un site industriel complet.

Le développement des technologies liées à Internet ont naturellement conduit au besoin d’évolution de Gédéon pour lui offrir une interface graphique web.

Cette interface web, totalement développée à l’aide de WebObjects, propose les mêmes facilités d’utilisation que la version client-serveur, partageant même le module d’affichage des documents transformé en composant ActiveX pour l’occasion. Elle cohabite avec la version client-serveur avec laquelle elle partage la base de documents.
